# AppAmigo

Aplicación móvil desarrollada con React Native y Expo que permite a los usuarios registrarse e iniciar sesión mediante Firebase Authentication.

---

## Avance del proyecto (Sprint 1)

En este primer sprint se desarrolló la base funcional de la aplicación utilizando Expo y React Native.

Se implementó un sistema de autenticación con Firebase que permite a los usuarios registrarse e iniciar sesión mediante correo electrónico y contraseña.

También se crearon tres pantallas principales que permiten la navegación dentro de la aplicación, mostrando el flujo básico de uso.

Adicionalmente, se configuró correctamente la conexión con Firebase y se aplicaron estilos personalizados a las vistas.

---

## Funcionalidades actuales
